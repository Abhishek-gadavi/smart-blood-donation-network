# 🩸 Smart Blood Donation Network

## 📌 Overview

Smart Blood Donation Network is a web-based platform that connects blood donors with patients in need. The system helps hospitals and patients quickly find blood donors and manage blood requests efficiently.

---

## 🚀 Features

### Donor Management
- Register Donors
- View Donors
- Search Donors by Blood Group
- Delete Donors

### Blood Request Management
- Create Blood Requests
- View Requests
- Priority Levels
  - Normal
  - Urgent
  - Critical
- Approve Requests
- Reject Requests
- Delete Requests

### Dashboard
- Total Donors Count
- Total Requests Count
- Approved Requests Count
- Rejected Requests Count
- Pie Chart Analytics

### Authentication
- Admin Login Page
